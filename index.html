Usage

    Simply enter the hints in the textboxes as they appear in the game.

    Words that matches the hints will be automatically shown in the results section.

    Out of the list, you should guess which one looks like the build first. Try all the words in the list and you'll end up guessing what it is.

    Use underscores to represent unknown characters.
×

About

    This solver is a word list cheat for all the themes in Guess The Build on Hypixel.

    At 90 seconds, hints are given out to all players.

    If you input this hint into the textbox, the solver will return a list of all the possible words.

    This website is licenced under AGPL-3.0 and is available in this GitHub repository.

    If a word is not on the list, open an issue here and I will add it.
